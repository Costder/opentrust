    Local Mode

    offline

    Run entirely on your machine — no cloud login required. Useful for air-gapped, privacy-sensitive, or offline deployments.

    In local mode the agent uses SQLite, local SMTP, and the local Kubo IPFS node. External APIs are disabled unless explicitly re-enabled.
