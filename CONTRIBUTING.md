# Contributing

OpenTrust accepts focused pull requests that improve passport schemas, registry APIs, CLI behavior, badge output, documentation, and payment contract definitions.

## Process

1. Open an issue or discussion for large changes.
2. Keep pull requests scoped.
3. Include tests for behavior changes.
4. Do not add real payment integrations, wallet code, Circle clients, Stripe clients, or secrets to this public repo.
5. Use clear review evidence for trust and security claims.

## Review Standards

Reviews are paid or credited for labor, not positive outcomes. Conflicts of interest must be disclosed. Auto-generated draft passports must never be presented as verified.
